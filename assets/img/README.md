# Imágenes

Coloque aquí las siguientes imágenes:

1. **logo-uct.png**: Logo de TEC-UCT (recomendado: 200x60px o similar)
   - Descargar desde: https://tec.uct.cl/

2. **favicon.png**: Favicon del sitio (recomendado: 32x32px o 64x64px)
   - Puede usar el mismo logo o crear un favicon personalizado

## Notas

- Los archivos deben estar en formato PNG
- Asegúrese de que los nombres de archivo coincidan exactamente con los referenciados en el código
- Si no tiene las imágenes, puede usar placeholders temporales
